0. 導入

Google Playストアより、当アプリ(QuickFlick Keyboard)をダウンロードしてください。
その後システム設定の中にある「キーボードを管理」を開き、当キーボードを有効化すると使えるようになります。

ちなみに各Androidバージョンにおけるシステム設定画面から「キーボードを管理」へのアクセス方法は以下の通りです。

  • Pie(Android 9): システム、言語と入力、仮想キーボード、キーボードを管理
  • Oreo(Android 8): Pieと同一手順
  • Nougat(Android 7): 言語とキーボード、仮想キーボード、キーボードを管理
  • Marshmallow(Android 6): Nougatと同一手順

1. 入力について

一般的な日本語入力アプリと同様、タップもしくはフリックで入力ができます。
同一の操作性をもつキーボードが他にないため、慣れないうちはプレビューを表示しながら使用することを強くおすすめします。
→ 入力について > ポップアップによるプレビュー
→ キーマップ

1.1 多段階フリック

QuickFlickではより少ないタップ数での入力を実現するため、多段階フリックを実装しています。
タップした箇所からの方向および段階的な距離によって入力する文字を変えることができます。
ほとんどのキーでは、タップのみで数字を、小距離でのフリックで大文字を、それ以上であれば小文字を入力できます。

なお、フリック感度は設定によって変えることができます。
詳しくはこちらをご確認ください。
→ 設定 > Horizontal/Vertical Flick Sensitivity

1.2 フリックインジケーター

フリックの状態(方向/距離)はキーボード上部のインジケーターによって随時確認できます。
インジケーターは方向を色で、距離はその濃度で表現します。

1.3 マルチタップによる入力制御

マルチタップにより、キー入力を制御することができます。
現在、以下の機能が利用可能です。

  • Flick reseter: マルチタップをする度、現在のフリック情報をリセットすることができます。
  • Input canceller: 2回マルチタップを行うことで、現在実行されようとしているキー入力をキャンセルすることができます。

それぞれの設定は設定画面から有効化/無効化ができます。
→ 設定 > Multi tap settings

1.4 ポップアップによるプレビュー

QuickFlickではフリック状態に応じた入力予定文字を可視化するプレビュー機能を利用することができます。
当アプリのプレビューは表示位置が他のキーボードアプリと異なり、なるべく指に隠れず、かつ他アプリの表示を一切邪魔しないよう常にキーボードに収まる最適な位置に表示されるような仕様になっており、その表示位置はタップ位置とFnキー(F1-3 F4-6 F7-9 Fa-cの4キー)の表示位置に依存します。後者についてはこちらもご覧ください。
→ Fnキー

またプレビュー機能は設定画面にて有効化/無効化が行えます。
→ 設定 > Input assistant

2. キーマップ

いくつかのキーを除き、当キーボードでは以下のような規則を満たすようにキーマップとキー印字が実装されています。

  1. 印字が1行のキーは印字内容をタップで入力可能にする
  2. 印字が2行のキーは上段に大きく表示されているものをフリックで、下段に小さく表示されているものをタップによって入力可能にする
  3. 上段印字の3文字目まではそれぞれ下/左/上へのフリックによって入力できる文字にする
  4. 上段印字の4文字目以降は右側は右への対応フリック距離が大きい順に並べる
  5. アルファベットのマッピングはフリック距離が大きくなると小文字になるようにする
  6. アルファベット入力の印字は大文字のみを表記する
  7. 括弧入力用キーの上段印字は法則3および4に従わず、キーマップ表にしたときに左に位置するものから順に並べる
  8. ひとつのキーに複数の記号入力を割り当てる場合、US配列キーボードの配置を参考にする

以下に上記規則に沿ったキーマップの例をいくつか示します。

上段印字: %^&*
下段印字: Space

キーマップ:

&
^ Space *
%

補足:

  • 各記号のフリック方向割り当てはUS配列キーボードの対応キーに準ずる
  • %: Shift + 5 キーで入力可能
  • ^: Shift + 6 キーで入力可能
  • &: Shift + 7 キーで入力可能
  • *: Shift + 8 キーで入力可能

上段印字: ABC-_
下段印字: 2

キーマップ:

c
C
b B 2 _ -
A
a

補足:

  • - および _ はUS配列キーボードにおいて同一キーに配置されている

上段印字: >]}
下段印字: )

キーマップ:

> ] } )

補足:

  • 右端に配置されることもあるキーなので、左フリックにまとめてマッピング
  • 使用率が比較的低い]を中途半端な位置に配置することによって>および}の相対的な入力しやすさを確保

実際に入力できる文字はプレビュー機能を用いることで随時確認することが可能です。

なお上記ルールに該当しない特殊なキーマップがいくつか存在します。

キー 入力方法 入力結果 備考
Esc Up/Down キーボードアプリ切り替え 下フリックのマッピングは廃止予定
Enter/Tab Down Enter Deleteキーとの誤タップ防止のため下フリックに配置
Up Shift-Enter 参考: Excelキーマップ

3. 方向キー

ArwあるいはPmvキーを一度もフリックせずにタップ入力すると、このキーの入力モードを切り替えることができます。
モードは以下の2種類があります。

  • Arw: 連続入力も可能な方向キーです。この状態ではフリックをするだけで入力が可能です。
  • Pmv: PageUp/Down, Home, Endのそれぞれを入力できます。

4. Fnキー

QuickFlickは左右どちらかにF1からF12までを入力できるFnキーを配置しています。
これらのキーにて上下どちらかにフリックするとFnキーの左右配置を入れ替えることができます。片手での操作を想定する当キーボードにおいて、この機能は左手もしくは右手向けへのレイアウト切り替えを実現するために実装されています。そのため前述のプレビュー機能は基本的にFnキーの表示位置に近くなるよう設計されています。

5. 修飾キー

QuickFlickはShift/Ctrl/Alt/Metaの4種類の修飾キーを実装しており、うち後者3つがキーボード上から操作することができます。
各修飾キーは3つのモードを持っています。

  • OFF: キー押下がされていない状態で、キー印字では小文字(c/a/m)で表現されます。
  • ON: キーが押下された状態で、キー印字では大文字(C/A/M)で表現されます。
    非修飾キーを入力するとOFFになります。
    ( <Ctrl>abc → <C-a> b c )
  • LOCK: キーが押下された状態で、キー印字では下線付きの大文字(C/A/M)で表現されます。
    非修飾キーを入力してもOFFになりません。
    ( <Ctrl-Lock>abc → <C-a> <C-b> <C-c> )

この3モードを切り替えるため、QuickFlickでは2種類のキーマップを用意しています。各モードへの対応と入力方法をまとめた表を以下に示します。

from OFF from ON from LOCK 入力方法
Mod-ON ON off off タップ
もしくは下フリック
Mod-LOCK LOCK LOCK off 右フリック
もしくは上フリック

現時点のキーボードではcaと印字されたキー(CtrlおよびAltが操作可能)とmaと印字されたキー(MetaおよびAltが操作可能)があり、CtrlおよびMetaがタップおよび右フリックで、Altが下あるいは上フリックでモード変更が可能です。ちなみにAltキーの状態は両キー連動しています。
またプレビュー表示では常に入力後のモードが表示されるようになっています。

6. 設定

現在、以下の設定が利用できます。

6.1 Horizontal/Vertical flick sensitivity

多段階フリックのフリック感度(閾値)を、thou単位(thou = ミリインチ)で設定できます。

  • 1st threshold: タップ/フリックの判定に使用される値です。この距離未満のフリックはタップとみなされます。
  • 2nd threshold: フリックレベルの判定に使用される値です。この値を超える度にフリックレベルが加算され、入力できる文字が変化します。

6.2 Multi tap settings

入力中にマルチタップを行った際の挙動を変更できます。
現在以下の機能を利用することができます。

  • Flick reseter
  • Input canceller

各機能の説明はこちらをご確認ください。
→ 入力について > マルチタップによる入力制御

6.3 Inupt assistant

入力を支援するための機能に関する設定を変更できます。

  • Use preview: プレビュー機能を使用します。

6.4 Appearance

キーボードの見た目を変更できます。

  • Indicator theme: フリックインジケーターの表示パターンを変更します。
  • Height in portrait: 端末が縦向きの時のキーボードの高さを変更できます。
  • Height in landscape: 端末が横向きの時のキーボードの高さを変更できます。
  • Footer in portrait: 端末が縦向きの時におけるキーボード下部にあるスペースの高さを変更できます。
  • Footer in landscape: 端末が横向きの時におけるキーボード下部にあるスペースの高さを変更できます。
  • Toggle layout to right: キーボードの左右寄せを切り替えます。この設定は前述のFnキーにおけるものと同一のものです。
