Explain how to use QuickFlick.

Download

QuickFlick(QuickFlick Keyboard) can download on Google Play.

Enable Keyboard

In Android 9 (Pie), Home Screen > Settings > System > Languages & input > Virtual Keyboard > Manage keyboards and turn switch at QuickFlick to ON.
Then you can see dialog, press 'OK'.

Following list is ways to access Manage keyboards settings in other version Android.

  • Android 6 (Marshmallow): Home Screen > Settings > Languages & input > Virtual Keyboard > Manage keyboards
  • Android 7 (Nougat): same to Marshmallow.
  • Android 8 (Oreo): same to Pie.

NOTE:
If you want to more infomation, search by Android(YourAndroidVersion) manage keyboards in Google.
