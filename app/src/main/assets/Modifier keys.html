Modifier keys

Now QuickFlick has 3 modifier key(Ctrl/Alt/Meta).
These keys can use in terminal emulator, code editor, and other apps which have any input field.

NOTE:
This app does NOT provide Cut/Copy/Paste/Redo/Undo functions.

Mode

Each Modifier keys have 3 modes.

  • OFF
    • default on start inputing.
    • do nothing.
  • ON
    • press modifier key once when you input char or number.
      (ex. <Ctrl:ON>abc > <C-a>bc)
  • LOCK
    • keep pressing modifire key.
      (ex. <Ctrl:LOCK>abc > <C-a><C-b><C-c>)

Control

There are 2 keymaps to control above modes.

from OFF from ON from LOCK How to input
Mod-ON ON off off Tap,
Down Flick
Mod-LOCK LOCK LOCK off Right Flick,
Up Flick

Check mode

If these keys turn ON, key label changes lower to upper.
(Meta: m-->M, Ctrl: c-->C, Alt: a-->A)

If LOCK mode, add underbar.
(ex. Ctrl-ON: C, Ctrl-LOCK: C)
